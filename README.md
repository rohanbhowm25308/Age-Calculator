# Age-Calculator
A modern and responsive Age Calculator that instantly calculates your exact age in years, months, and days from your date of birth. Built with HTML, CSS, and JavaScript, featuring a clean UI and real-time age calculation.

#  Age Calculator

A simple, modern, and user-friendly Age Calculator built using **HTML, CSS, and JavaScript**. This application allows users to calculate their exact age based on their date of birth, displaying results in years, months, and days instantly.

##  Features

* 🎯 Accurate age calculation
* 📅 Date of Birth selection
* ⚡ Instant result generation
* 🎨 Clean and responsive user interface
* 📱 Mobile-friendly design
* 🚀 Fast and lightweight
* 🔄 Real-time calculations
* 💻 Built with pure HTML, CSS, and JavaScript

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
